About the detector

This project uses Natural Language Processing (NLP) with TF-IDF features and a Logistic Regression classifier trained on labeled datasets of real and fake news.

  • Text cleaning with stopword removal and lemmatization
  • TF-IDF vectorization (unigrams and bigrams)
  • Logistic Regression classification
  • Confidence score per prediction
