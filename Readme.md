# Fake News Detector

A machine learning-based fake news detection app built with Streamlit. This app uses NLP and Logistic Regression to classify news as FAKE or TRUE in real-time. It includes sample headlines from the dataset used to train the model, allowing users to choose a headline and get an analysis with an output.

## Features

- Real-time classification of news as FAKE or TRUE
- Sample headlines from the training dataset
- Interactive prediction with confidence scores

## Installation

1. Clone the repository: `git clone https://github.com/your-username/fake-news-detector.git`
2. Install the required dependencies: `pip install -r requirements.txt`

## Usage

1. Run the app: `streamlit run app.py`
2. Choose a headline from the sample headlines section
3. Click the "Detect News" button to get the analysis and output
